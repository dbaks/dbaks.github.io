创建自己的Pawn

发表于 2021-03-17

概述 ： 描述了如何从零创建一个Pawn

导航页面

创建一个空的Pawn并绑定基础操作

[NOTE] Pawn是一种由真实玩家或AI控制的 Actor。

Unreal 文档

  1. 向项目添加自定义 c++ Pawn 类，并命名为MyPawn
  2. 初始化MyPawn，并添加一些组件使MyPawn可见。

    场景组件（SceneComponent) 具有变形和支持附件功能，但没有渲染或碰撞功能。
    它在层级中用作”虚拟”组件来偏移其他组件，或者与其子组件交互以使其像一个整体。

  3. 在编辑器的项目设置中绑定操作，并且为SceneComponent的StaticMesh选择资产
  4. 定义并创建操作对应的函数与将要使用的变量

    [^NOTE] : 使用 FMath::Clamp 约束输入中得到的值，将其约束在-1到+1的范围内。

  5. 绑定函数并将代码添加到AMyPawn::SetupPlayerInputComponent 中

    [^NOTE] : 关于SetupPlayerInputComponent，未完成
